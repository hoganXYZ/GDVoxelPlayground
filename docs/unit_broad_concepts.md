# Unit
This is a base description of a unit in the ant game. It's anything that moves, has health, can attack, be controlled, etc.

## Categories
- Class: Animal
- Type: Crawling, Flying, Slithering, Growing
- Family: Ant, Bee, Worm, etc


## Plan
Ideally I don't want to overdefine this yet, I want to build components that I can add onto units so I can play with capabilities.
This reminds me of the ECS research I did, this would probably be a good use case for that.
